# text-cornucopia-cli

> generate a bunch of strings from a list using a pattern

cli from text cornucopia tool https://hsalmeida.github.io/text-cornucopia/

## Highlights
- cli commands
- multi outputs types
- redis support (beta)

## Install

```console
$ npm install -g cornucopia
```

## Usage

```console
$ cornucopia
```
